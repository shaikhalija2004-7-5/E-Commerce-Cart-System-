# 🛒 E-Commerce Cart System (Python + Excel)

## 📌 Overview

This project is a simple **E-Commerce Cart System** built using Python. It allows users to select products, add them to a cart, and generate a final bill. The system reads product data from an Excel file and updates stock after purchase.

## 🚀 Features
* 📂 Read product data from Excel file
* 🛍️ Add items to cart
* 🔢 Calculate total quantity and price
* 📉 Update stock after purchase
* 🧾 Generate bill summary
* ❌ Input validation for better user experience

## 🛠️ Technologies Used

* Python
* File Handling
* Excel Integration using `openpyxl`
* Object-Oriented Programming (OOP)

## 📁 Project Structure

Ecommerce-System/
│── main.py
│── goods_list.xlsx
│── README.md

## ⚙️ How It Works

1. The program reads product data from the Excel file.
2. User selects products by entering product ID.
3. Quantity is entered and validated.
4. Cart stores selected items.
5. Total amount is calculated.
6. Stock is updated in the Excel file after checkout.

## ▶️ How to Run

1. Install required library:
pip install openpyxl
2. Run the program:
python main.py
